Présentation

Blue est un petit logiciel permettant d'automatiser les mises sur les enchères horaires du jeu Darkorbit. En effet, après vous être connecté au jeu avec cet outil, vous aurez la possibilité de choisir les objets que vous souhaitez acquérir. Le logiciel se chargera ensuite de miser une somme fixée de crédits sur ces objets chaque heure. Vous devez définir le temps entre la fin des enchères et la mise, sachez que toutes les mises doivent être espacées d'au moins 10 secondes entre elles pour assurer le bon fonctionnement du logiciel.

Partie technique

Ce logiciel a été fait en C++ avec le framework Qt pour gérer simplement l'interface utilisateur et la communication avec le serveur en https. Ce logiciel est donc multiplateforme en théorie mais je n'ai publié qu'une version Windows car c'est l'OS que j'utilise. Je ne conserve aucune information relative à votre compte.

Note : Étant donné que cet outil est un logiciel de triche je ne le distribue par sur ce site. Toutefois, s'il vous intéresse, vous pouvez me le demander en mail.
